---

## Overview

Wander.pk is a simulated travel booking platform designed for the Pakistani market. Users can browse, select, and book travel services across multiple categories with shopping cart, user authentication, booking management, and receipt generation — all using `localStorage`.

| Metric | Value |
|--------|-------|
| Services | 4 Categories |
| Flights | 6 Routes |
| Tours | 12 Packages |
| Hotels | 6 Properties |
| Buses | 6 Routes |

## Features

| Category | Features |
|----------|----------|
| ✈️ Flights | 6 routes, class selection, seat grid, live tracking |
| 🏔️ Tours | 12 destinations, filters, pagination, package variants |
| 🏨 Hotels | 6 properties, room types, ratings, amenities |
| 🚌 Buses | 6 routes, seat classes, operators, live tracking |
| 🛒 Cart | Add/remove, quantity adjust, persistent storage |
| 👤 Auth | Signup/Login, admin access, session persistence |
| 💳 Checkout | Multiple payments, Pay Later, receipts, print |
| 🔍 Search | Keyword, location, price, duration filters |
